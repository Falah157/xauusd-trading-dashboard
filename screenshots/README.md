# Screenshots Directory

This directory contains screenshots of the Smart Gold Trading Dashboard.

## How to Take Screenshots

1. **Start the application**:
   ```bash
   streamlit run main.py
   ```

2. **Navigate to** `http://localhost:8501`

3. **Take screenshots of these sections**:
   - Dashboard Overview (main page with charts)
   - Technical Analysis (indicators section)
   - AI Predictions (prediction section)
   - Authentication (login page)

## Screenshot Guidelines

- Use high resolution (1920x1080 or higher)
- Capture the full browser window
- Make sure the dashboard is fully loaded
- Use a clean browser without bookmarks bar
- Consider using dark mode for better appearance

## File Naming Convention

- `dashboard-overview.png` - Main dashboard
- `technical-analysis.png` - Technical indicators
- `ai-predictions.png` - AI predictions section
- `authentication.png` - Login page

## Tools for Screenshots

- **Mac**: Cmd + Shift + 4 (area selection) or Cmd + Shift + 3 (full screen)
- **Windows**: Snipping Tool or Win + Shift + S
- **Linux**: gnome-screenshot or similar tools
- **Browser**: Right-click → "Inspect" → Device toolbar for mobile screenshots

## Adding to README

After taking screenshots, update the README.md file to reference them correctly:

```markdown
![Dashboard Overview](screenshots/dashboard-overview.png)
![Technical Analysis](screenshots/technical-analysis.png)
![AI Predictions](screenshots/ai-predictions.png)
```
